## Run Locally
[Live](https://gh-repo-search-ideo.netlify.app/users/yunheli)

Clone the project


```bash
git clone https://github.com/Nightmare9254/gh-search-ideo.git
```

Go to the project directory

```bash
cd gh-search-ideo
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```
